# Task 1: Simple E-commerce Store - CodeAlpha Internship

## Project Description
A basic e-commerce web application built as Task 1
for CodeAlpha Internship. Users can browse products, add to cart, and place orders.

## Tech Stack
Frontend: HTML5, CSS3, JavaScript
Backend: Node.js with Express.js - runs on localhost:3000
Database:In-Memory array for storing products, users, and orders

## Features Implemented
- User registration and login
- Product listings page
- Shopping cart functionality
- Order processing system
- Temporary data storage using JavaScript arrays

## How to Run Locally
1. Clone this repository
2. Navigate to Task1 folder
3. Install dependencies: `npm install`
4. Start server: `node server.js` or `npm start`
5. Open browser: `http://localhost:3000`
